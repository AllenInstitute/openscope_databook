Reading an NWB File#

After downloading an NWB file, you may want to view the data inside. You can do basic reading of the file with PyNWB. This is a package designed to utilize, modify, and process NWB files. The basic read functionality of PyNWB is shown below. This notebook is intended to explore the NWB file which was specified for download in Downloading an NWB file. If you choose a different file, make sure it’s already downloaded!

Environment Setup#

from pynwb import NWBHDF5IO

Reading an NWB File#

You can read in a PyNWB file with NWBHDF5IO to retrieve an io object. You can use the .read method to actually read it in. From there, you can see the raw data of the NWB file and can print the fields you are interested in.

nwb_filepath = "./sub-699733573_ses-715093703.nwb"

io = NWBHDF5IO(nwb_filepath, mode="r", load_namespaces=True)
nwb = io.read()

print(nwb)
